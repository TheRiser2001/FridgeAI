🧠 FridgeAI – Deine intelligente Rezept-App

FridgeAI ist eine iOS-App, mit der du basierend auf deinen vorhandenen Zutaten passende Rezepte entdecken kannst. Einfach Zutaten einscannen oder manuell eintragen – und die App schlägt dir leckere Rezepte vor. Perfekt für alle, die kreativ kochen oder Lebensmittelverschwendung vermeiden möchten.

🚀 Features

📷 Zutaten-Scanner: Nutze einen QR-/Barcode-Scanner, um Zutaten schnell zu erfassen.
🧑‍🍳 Rezepte entdecken: Erhalte Rezeptvorschläge, basierend auf den vorhandenen Zutaten.
🌍 Sprachübersetzung: Englische Rezepte werden automatisch ins Deutsche übersetzt.
🔐 Benutzer-Authentifizierung: Anmeldung via E-Mail durch Firebase.
🛠️ Tech-Stack

Programmiersprache: Swift
UI-Framework: SwiftUI
Plattform: iOS
🌐 Verwendete APIs

🥘 Spoonacular API – Für Rezeptvorschläge und Zutatenerkennung
🔐 Firebase Authentication – Für sichere E-Mail-Login-Funktion
🌐 DeepL API – Für die automatische Übersetzung von Rezepten
